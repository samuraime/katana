# Katana [![CircleCI](https://circleci.com/gh/samuraime/katana.svg?style=shield)](https://circleci.com/gh/samuraime/katana)

the backend of suzaku

## Scripts

  - `yarn run dev` 同时运行前后端
  - `yarn run be` 开发环境后端
  - `yarn run fe` 开发环境前端
  - `yarn run start` 生产环境后端
  - `yarn run deploy` 部署前后端
